Cameron has photorealistic reference images (generated by Grok) showing the exact target look: white pearl egg with iridescent rim band, fine surface scratches, crystalline micro-texture. Saved in repo root as `grok-image-*.png`. Favorite is `grok-image-21a2dae3-*.png`.

**Hybrid approach planned:** Use a Grok reference image as a pre-baked texture overlay for static surface detail (scratches, organic iridescent band flow, micro-structure). Shader handles dynamic elements (specular tracking mouse, animated pulse). Sample texture for base color + iridescence detail, ADD dynamic specular on top.

**Key observations from references:**
- Iridescence is a concentrated RIM BAND (not gradual center-to-edge blend)
- Center is almost pure clean white/cream
- Rainbow sequence: pink → magenta → cyan → blue → gold around rim
- Fine surface scratches / crystalline micro-texture (not reproducible procedurally)
- Clean punchy specular highlight in upper region

**How to apply:** This is a Sprint 4 (Visual Polish) task. For now, keep the procedural shader working but plan the texture pipeline. Don't over-invest in procedural iridescence tuning — the texture will carry most of the visual weight.
